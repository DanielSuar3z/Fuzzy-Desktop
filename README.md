Este proyecto presenta el desarrollo de un sistema inteligente basado en lógica difusa para evaluar la postura corporal humana en función de tres variables principales: el ángulo del cuello, la inclinación de la espalda y la alineación de las piernas. El objetivo es clasificar el estado postural de una persona en categorías como excelente, buena, regular, mala o crítica. La lógica difusa permite una interpretación flexible de estas variables, considerando valores continuos y rangos superpuestos
